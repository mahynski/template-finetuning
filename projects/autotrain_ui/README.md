# How To

1. Install the virtual environment (only needs to be done once)
~~~bash
$ uv sync
~~~

2. Install additional tokenizers (only needs to be done once)
~~~bash
$ bash setup.sh
~~~

3. Run the AutoTrain UI - this invokes the virtual environment automatically
~~~bash
$ bash start_autotrain_ui.sh
~~~
